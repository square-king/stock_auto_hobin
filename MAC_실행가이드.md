# 🤖 Mac 자동매매 봇 실행 가이드

## 📌 빠른 시작 (가장 쉬운 방법)

### 1. 터미널에서 바로 실행
```bash
cd /Users/jiho_dt/Desktop/stock_auto_test
python3 main.py
```

### 2. 더블클릭으로 실행
Finder에서 다음 파일을 더블클릭:
```
/Users/jiho_dt/Desktop/stock_auto_test/scripts/start_trading.command
```

## 🔧 Mac 설정 (권장)

### 1. 슬립 방지 설정
**시스템 설정 > 에너지 > 디스플레이가 꺼져도 자동 잠자기 방지** 체크

또는 터미널에서:
```bash
# 수동 슬립 방지 (터미널 열려있는 동안 유지)
caffeinate -i
```

### 2. 화면보호기 비활성화
**시스템 설정 > 화면 잠금 > 사용 안 함**

## 📅 봇 실행 스케줄

| 시간       | 작업                               |
| ---------- | ---------------------------------- |
| **09:01**  | 전일 시그널 → 매수 주문 실행       |
| **매 5분** | 보유 종목 청산 시그널 체크         |
| **15:35**  | 전략별 스크리닝 + 익일 시그널 스캔 |
| **15:40**  | 일일 리포트 카카오톡 발송          |
